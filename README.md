# XML Calabash 2.x binary metadata extraction extension step

[![Build Status](https://travis-ci.org/ndw/xmlcalabash2-metadata-extractor.svg?branch=master)](https://travis-ci.org/ndw/xmlcalabash2-metadata-extractor.svg?branch=master)

This repository contains an
[XML Calabash](http://github.com/ndw/xmlcalabash2) 2.x extension
step that can extract metadata from binaries. It uses
[MetadataExtractor](https://drewnoakes.com/code/exif/) to extract
data from images. It uses [PDFBox](https://pdfbox.apache.org/) to extract
data from PDF files.
